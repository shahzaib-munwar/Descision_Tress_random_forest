DECISION-TREES-AND-RANDOM-FORESTS
--
USED SKLEARN'S BUILT-IN WINE DATASET APPLIED EDA, 

-TRAINED MULTIPLE CLASSIFICATION MODELS, 

PERFORMED HYPERPARAMETER TUNING AND 

-VISUALIZED THE RESULTS DEEPLY

LIBRARIES USED:
--
NUMPY

PANDAS

MATPLOTLIB

SEABORN

SCIKIT-LEARN

ABOUT DATASET
--
IT IS THE WINE DATASET FROM SKLEARN WHICH INCLUDES 178 SAMPLES WITH 13 CHEMICAL FEATURES USED TO CLASSIFY WINES INTO 3 CLASSES (CLASS_0, CLASS_1, CLASS_2)

EDA
--
APPLIED DIFFERENT PANDAS FUNCTIONS TO EXPLORE THE DATASET, MAPPED TARGET LABELS TO MEANINGFUL CLASS NAMES, ANALYZED CLASS DISTRIBUTION USING VALUE COUNTS AND VISUALIZED THE BALANCE ACROSS CLASSES

ABOUT MODEL
--
IMPORTED DECISIONTREECLASSIFIER AND RANDOMFORESTCLASSIFIER FROM SCIKIT-LEARN

TRAINED A FULL DECISION TREE (NO DEPTH LIMIT) AND A SIMPLE DECISION TREE (MAX DEPTH = 3)

FOUND OPTIMAL TREE DEPTH BY TESTING DEPTHS FROM 1 TO 15 AND COMPARING TRAIN VS TEST ACCURACY

TRAINED A DEFAULT RANDOM FOREST WITH 100 ESTIMATORS

APPLIED GRIDSEARCHCV FOR HYPERPARAMETER TUNING ACROSS N_ESTIMATORS, MAX_DEPTH, MIN_SAMPLES_SPLIT AND MIN_SAMPLES_LEAF

EVALUATED ALL MODELS USING ACCURACY SCORE, CLASSIFICATION REPORT AND CONFUSION MATRIX
